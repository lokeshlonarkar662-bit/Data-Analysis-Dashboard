🛒 Blinkit Grocery Sales Analysis Dashboard (Power BI)
📌 Project Overview

This project focuses on analyzing Blinkit grocery sales data using Microsoft Power BI to generate meaningful business insights. The dashboard transforms raw sales data into interactive visualizations that help businesses understand product performance, outlet sales trends, and customer ratings.

The objective of this project is to demonstrate how data analytics and visualization tools can support data-driven decision making in the retail and e-commerce sector.

🎯 Problem Statement

Online grocery platforms generate a massive amount of sales data every day. Analyzing this data manually is difficult and inefficient.

Businesses often struggle to:

Track product performance

Monitor outlet sales trends

Analyze customer ratings

Identify patterns in sales data

Without proper analytics tools, it becomes challenging to extract insights and make strategic business decisions.

💡 Proposed Solution

The proposed solution is an interactive Power BI dashboard that analyzes Blinkit grocery sales data and presents insights visually.

The system performs the following operations:

Import grocery sales dataset from Excel

Clean and transform data using Power Query

Create key performance indicators using DAX

Build interactive visualizations

Provide filters for dynamic data analysis

This solution helps businesses understand sales patterns and improve decision making.

🛠️ Technologies Used
Tools

Microsoft Power BI

Microsoft Excel

Features / Libraries

Power Query – Data cleaning and transformation

DAX (Data Analysis Expressions) – KPI calculations

Power BI Visualizations – Interactive charts and dashboards

⚙️ System Development Approach

The development process includes the following steps:

Import the dataset from Excel into Power BI.

Clean and transform the data using Power Query.

Create calculated measures and KPIs using DAX formulas.

Build interactive visualizations and dashboards.

Deploy the dashboard using Power BI Service.

📊 Key Performance Indicators (KPIs)

The dashboard calculates the following KPIs:

Total Sales

Average Sales

Number of Items Sold

Average Customer Rating

These KPIs provide a quick overview of the business performance.

📈 Dashboard Visualizations

The dashboard contains multiple visualizations, including:

Donut Chart – Sales by Fat Content

Bar Chart – Sales by Outlet Establishment Year

Funnel Chart – Outlet Location Performance

Item Type Analysis – Sales by different product categories

Outlet Size Comparison

Interactive Filters – For outlet size and item type

These visualizations help identify patterns and trends in grocery sales.
